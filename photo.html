Foto collage

Hier leer je CSS te gebruiken om HTML-elementen exact te positioneren en een foto collage te maken.

  • Voeg een div toe aan je pagina en plaats er zoveel afbeeldingen in als je wil. Geef de div en img elementen id waarden.
<div id="photoBox" class="relPos">
<img id="imgBurcht" class="absPos" src="burcht.jpg" alt="De Burcht" />
<img id="imgRondvaart" class="absPos" src="rondvaart.jpg" alt="Bootje varen..." />
</div>

De foto's verschijnen na elkaar op de webpagina, in de volgorde waarin ze in je code staan.

  • Voeg in je CSS bestand de volgende CSS class toe voor de elementen in de div:
.absPos {
position: absolute;
 }
  • Vervolgens moet je de eigenschap position: relative; toevoegen aan de container zelf en er een maat voor opgeven. Dit zorgt ervoor dat de posities van de andere elementen relatief ten opzichte van (dat wil zeggen, binnen) de container worden gedefinieerd.
.relPos {
position: relative;
 }
#photoBox {
width: 800px;
 height: 400px;
 }
  • Maak vervolgens een set stijlregels voor elk van de elementen met behulp van id selectors om hun afmetingen (width en/of height eigenschappen) en hun exacte posities in te stellen.

Om de positie van een element te definiëren, zijn er vier eigenschappen die je kan gebruiken: left, right, top en bottom. Ze geven aan hoe ver elke randen van de rand van de parent moet zijn. Gebruik top of bottom voor de verticale positie en left of right voor de horizontale positie.


  • Kies exacte posities voor elk van je afbeeldingen en gebruik een van de eigenschappen left, right, top of bottom om die posities in je CSS regels te definiëren. Deze code plaatst de afbeelding van de Burcht bijvoorbeeld 100 pixels van boven en 60 pixels van links:
#imgBurcht {
width: 250px;
 top: 100px;
 left: 60px;
 }

Opmerking: de positiewaarden kunnen ook negatief zijn! Als je een negatieve waarde gebruikt, wordt het element buiten de container geduwd, over de rand die je hebt opgegeven.

Dingen laten overlappen

Misschien wil je dat sommige afbeeldingen elkaar overlappen. Maar hoe kies je welke bovenaan staat?

  • Kies twee afbeeldingen en geef ze posities waardoor ze elkaar overlappen.
  • Voeg een extra eigenschap toe: z-index: 10; aan een van hen, en voeg dan z-index: 7; toe aan de ander.
  • Bekijk het resultaat op je webpagina.

  • Verwissel nu de waarden van z-index, zodat de 7 en de 10 andersom staan. Zie je het verschil op je webpagina?

Met de eigenschap z-index kan je bepalen hoe twee of meer elementen elkaar moeten overlappen. De waarde kan elk geheel getal zijn.

Het element met het hoogste nummer komt bovenop de stapel, of met andere woorden helemaal vooraan. Het element met het volgende hoogste nummer bevindt zich daarachter en vóór de anderen, enzovoort, totdat je bij het element met het laagste nummer komt, dat achteraan achter alle andere elementen verschijnt.

Je kan elk HTML-element op deze manier plaatsen, niet alleen afbeeldingen. Je kan bijvoorbeeld een p element gebruiken om wat tekst over een foto toe te voegen.

Uitdaging: maak een fotocollage

Probeer je eigen fotocollage te maken zoals hieronder afgebeeld! Gebruik exacte positionering samen met verschillende waarden voor z-index om het overlappende effect te krijgen zoals je het wil.

Hieronder staat de HTML-code voor de fotocollage op de voorbeeld website. Er zijn vijf foto's en een stuk tekst allemaal in een div.

<div id="photoBox" class="relPos">
<img id="imgBurcht" class="collagePhoto absPos" src="burcht.jpg" alt="De Burcht" />
<img id="imgRondvaart" class="collagePhoto absPos" src="rondvaart.jpg" alt="Bootje varen..." />
<img id="imgNaturalis" class="collagePhoto absPos" src="naturalis.jpg" alt="Dino's in Naturalis" />
<img id="imgOudheden" class="collagePhoto absPos" src="mummy.jpg" alt="Mummies in Rijksmuseum voor Oudheden" />
<img id="imgVliet" class="collagePhoto absPos" src="zwembad.jpg" alt="Zwemmen in De Vliet!" />
<p id="photoTekst" class="absPos">Leiden!</p>
</div>

En dit zijn de CSS regels die de posities bepalen voor elk van de foto's in de collage:

#imgBurcht {
width: 120px;
 top: 130px;
 left: 70px;
 z-index: 10;
 }
#imgRondvaart {
width: 250px;
 top: 210px;
 left: 120px;
 z-index: 7;
 }
#imgNaturalis {
width: 150px;
 top: 240px;
 left: 325px;
 z-index: 9;
 }
#imgOudheden {
width: 110px;
 top: 215px;
 left: 25px;
 z-index: 11;
 }
#imgVliet {
width: 250px;
 top: 90px;
 left: 170px;
 z-index: 6;
 }
#photoTekst {
font-family: "brush script MT";
 color: blue;
 font-size: 4em;
 left: 20px;
 top: 210px;
 z-index: 20;
 }

En dit zijn de andere CSS classes die zijn gebruikt:

.collagePhoto {
border: 1px solid white;
 }
.relPos {
position: relative;
 }
.absPos {
position: absolute;
 }
